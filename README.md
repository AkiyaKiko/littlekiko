### Usage
Step 1. 
Edit the ```/Modules/__init__.py```

Step 2. 
Change the ```Token``` into your own Telegram Bot Token and save the file

Step 3. 
Run the command below
```
pip install -r requirements.txt
```
```
python main.py
```
OR (Based on your System)
```
python3 main.py
```
